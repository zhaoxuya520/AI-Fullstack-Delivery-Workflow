# Task Input Template

用于向总控提交任务时统一描述输入。

## 任务目标

我要完成什么？

```text
例如：实现用户登录功能 / 修复线上接口超时 / 做上线前安全检查
```

## 当前上下文

项目背景、已有代码、已有文档、技术栈。

```text
项目类型：
技术栈：
相关目录：
当前状态：
```

## 期望输出

希望得到什么产物？

```text
代码 / 文档 / 报告 / 测试用例 / 部署方案 / 安全评审 / 数据分析
```

## 限制条件

时间、环境、权限、技术栈、安全边界。

```text
时间限制：
运行环境：
权限边界：
不能改动：
必须兼容：
```


```text

```
