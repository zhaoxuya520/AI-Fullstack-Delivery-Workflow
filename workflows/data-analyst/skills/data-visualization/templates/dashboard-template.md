# 仪表盘设计模板

## 基本信息
```text
仪表盘名：
受众：PM / 运营 / 管理层
数据源：
刷新频率：实时 / 每小时 / 每日
工具：Metabase / Grafana / Looker / Tableau
```

## 布局

| 区域 | 内容 | 图表类型 |
|---|---|---|
| 顶部 KPI | DAU / GMV / 转化率 | 大数字 + 趋势 |
| 趋势 | 7日 / 30日趋势 | 折线图 |
| 漏斗 | 注册→下单→支付 | 漏斗图 |
| 分布 | 用户来源 / 地域 | 饼图 / 柱状图 |
| 明细 | Top 10 商品 | 表格 |

## 图表选型规则

```text
趋势（时间序列）→ 折线图
对比（分类对比）→ 柱状图
占比（构成）→ 饼图 / 环形图
分布（数值分布）→ 直方图
关系（两变量）→ 散点图
漏斗（转化）→ 漏斗图
地理（地域）→ 地图
排名（Top N）→ 条形图
```

## 自检

```text
□ 受众明确
□ KPI 突出
□ 图表类型合适
□ 不超载（< 10 图）
□ 有时间筛选
□ 有对比基线
□ 移动端可看
```
