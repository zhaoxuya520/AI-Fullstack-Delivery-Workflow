# 补充反同质化和品类适配风格原则

## Date

2026-05-18

## Workflow

ui-ux-designer

## Task Background

用户强调：避免 AI 化风格不等于缺少创新，也不能把所有类型页面统一成纯白企业风。例如动漫相关产品不应被设计成普通企业 SaaS 页面。

## Problem

上一轮已加入反 AI 化视觉检查，但如果执行过度，可能把“去掉 AI 味”误解为“去掉风格表现”，最终导致所有产品同质化，特别是动漫、游戏、IP、潮流社区等内容型或文化圈层强的产品。

新增重点：

```text
反 AI 化不是去风格化。
反 AI 化不是保守化。
反 AI 化不是统一成纯白企业 SaaS。
风格必须匹配产品品类、用户圈层、内容气质和品牌调性。
动漫、游戏、IP、潮流、创意产品可以有更强表现力，但要有业务来源、组件落地和可访问性兜底。
```

新增风格模式：

```text
动漫 / IP / ACG 风格
游戏 / 沉浸式活动风格
```

## Reusable Lesson

避免 AI 味的核心是避免无意义模板化，而不是消灭风格。真实产品的好设计应同时具备：品类识别度、用户文化适配、业务任务清晰、组件系统可落地、状态完整和可访问性。
