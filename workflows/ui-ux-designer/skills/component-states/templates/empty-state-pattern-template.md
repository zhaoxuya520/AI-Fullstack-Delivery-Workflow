# 空状态模式模板

## 1. 空状态类型

```text
页面/组件：
空状态类型：首次使用为空 / 搜索无结果 / 权限为空 / 错误导致为空 / 功能未启用
目标用户：
```

## 2. 内容结构

```text
标题：
说明：
主行动：
次行动：
辅助链接：
是否需要插图：是 / 否
```

## 3. 状态规则

| 类型 | 原因 | 推荐主行动 | 说明 |
|---|---|---|---|
| 首次使用为空 | 没有创建数据 | 创建 / 导入 / 查看示例 | 引导开始 |
| 搜索无结果 | 条件过窄 | 清除筛选 | 保留搜索上下文 |
| 权限为空 | 无查看权限 | 申请权限 | 避免暴露敏感信息 |
| 错误导致为空 | 加载失败 | 重试 | 显示可恢复动作 |
| 功能未启用 | 配置缺失 | 去配置 | 说明启用条件 |

## 4. 验收标准

```text
用户知道为什么为空。
用户知道下一步能做什么。
主行动与空状态原因匹配。
空状态不会误导用户以为系统出错。
权限空状态不会泄露敏感信息。
```
