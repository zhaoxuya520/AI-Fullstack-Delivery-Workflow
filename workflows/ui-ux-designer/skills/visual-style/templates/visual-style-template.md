## 13. 验收标准

```text
1. 风格能覆盖主要页面和核心组件。
2. 风格能覆盖加载、空、错误、权限、成功状态。
3. 页面去掉装饰后仍能完成主要任务。
4. 使用真实业务数据后布局仍成立。
5. 前端能用现有组件库或明确新增组件实现。
6. 不出现明显 AI 模板感。
```
