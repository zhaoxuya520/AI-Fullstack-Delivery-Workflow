# 用户流程模板

## 1. 流程目标

```text
流程名称：

```

## 2. 主流程

```mermaid
flowchart TD
    A[用户进入] --> B[查看核心信息]
    B --> C[执行主操作]
    C --> D{是否成功}
    D -->|是| E[展示成功反馈]
    D -->|否| F[展示错误原因和恢复动作]
```

