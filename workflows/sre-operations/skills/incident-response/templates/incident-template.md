# 事故响应记录模板

## 事故信息
```text
事故 ID：INC-YYYY-MM-DD-XXX
严重度：P1 / P2 / P3 / P4
状态：进行中 / 已恢复 / 已关闭
影响：[用户数 / 功能 / 区域]
开始时间：
恢复时间：
持续时间：
Incident Commander：
```

## 时间线
| 时间 | 事件 | 操作人 |
|---|---|---|
| HH:MM | 告警触发 | 系统 |
| HH:MM | IC 确认 | [人] |
| HH:MM | 止血操作 | [人] |
| HH:MM | 服务恢复 | [人] |

## 影响评估
```text
受影响用户：
受影响功能：
数据影响：
财务影响：
```

## 止血措施
```text
□ [措施 1]
□ [措施 2]
```

## 根因（初步）
```text
直接原因：

```

## 后续
```text
□ 复盘安排：[日期]
```
