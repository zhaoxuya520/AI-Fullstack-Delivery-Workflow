# 备份恢复与数据操作安全指南

## 操作前原则

```text
确认授权
→ 确认环境
→ 确认影响面
→ 确认备份
→ 确认回滚
→ 小批量验证
→ 执行并记录
```

## 备份检查

- 备份是否覆盖目标表和相关依赖表。
- 备份是否在可接受恢复点内。
- 是否验证过恢复流程，而不只是确认备份文件存在。
- 是否有恢复后的数据一致性检查。

## 批量操作检查

- WHERE 条件是否经过复核。
- 是否先 SELECT 预估影响行数。
- 是否分批、限速、可暂停。
- 是否记录执行进度和失败项。
- 是否避免输出敏感数据。

## 数据导出边界

- 明确授权和用途。
- 最小字段集导出。
- 敏感字段脱敏。
- 限制保存位置和保留时间。
- 记录销毁或归档方式。
