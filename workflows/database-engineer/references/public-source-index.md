# 数据库工程师公开资料索引

## 官方文档

- PostgreSQL Documentation — 数据类型、约束、索引、事务、分区、执行计划。
- MySQL Documentation — InnoDB、Online DDL、索引、事务、复制和备份。
- SQLite Documentation — 类型系统、迁移限制、并发和事务。
- Redis Documentation — 数据结构、持久化、过期策略和内存边界。

## 方法论主题

| 主题 | 参考方向 |
|------|----------|
| 数据建模 | 规范化、关系建模、领域实体、生命周期 |
| 约束设计 | 主键、外键、唯一约束、检查约束、空值语义 |
| 索引设计 | 选择性、复合索引、覆盖索引、执行计划 |
| 数据迁移 | expand/contract、在线迁移、回填、回滚 |
| 查询优化 | EXPLAIN、慢日志、N+1、分页、聚合 |
| 操作安全 | 备份恢复、变更窗口、脱敏、权限审计 |

## 使用原则

- 优先查目标数据库版本对应的官方文档。
- 不把某一数据库的特性默认套用到另一数据库。
- 公开资料只能作为方法参考，生产参数以项目实际环境为准。
