# 交付门禁 Checklist

## 1. 需求门禁

- [ ] PRD 已确认
- [ ] MVP 和非目标范围明确
- [ ] 用户故事已拆分
- [ ] 验收标准可测试
- [ ] 角色权限和异常场景明确
- [ ] 安全/合规约束已识别

## 2. 设计门禁

- [ ] UI/UX 关键页面和状态已确认
- [ ] API 契约已确认
- [ ] 数据实体和生命周期已确认
- [ ] 权限模型已确认
- [ ] 错误码和异常处理已确认

## 3. 开发门禁

- [ ] 核心功能完成
- [ ] 单元测试或基础验证通过
- [ ] 前后端联调通过
- [ ] 关键异常路径已处理
- [ ] 代码或配置没有明显安全风险

## 4. 测试门禁

- [ ] QA 测试通过
- [ ] 回归测试通过
- [ ] 高优先级 Bug 已修复
- [ ] 已知问题记录清楚
- [ ] 安全检查通过或风险已接受

## 5. 上线门禁

- [ ] 部署说明就绪
- [ ] 回滚方案就绪
- [ ] 环境变量和密钥配置确认
- [ ] 监控和日志确认
- [ ] 健康检查通过
- [ ] 相关方已通知

## 6. 文档和经验门禁

- [ ] 用户/团队文档已更新
- [ ] API 或部署文档已更新
- [ ] 项目复盘已完成
- [ ] 可复用经验已写入 field-journal
