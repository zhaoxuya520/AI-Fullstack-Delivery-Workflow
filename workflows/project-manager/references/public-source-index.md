# 项目经理工作流公开资料索引

本索引用于记录项目经理工作流可参考的公开资料来源。沉淀方法、结构和检查点，不复制受版权保护的大段原文。

## 项目管理基础

| 来源 | 主题 | 可复用要点 |
|---|---|---|
| [PMBOK Guide - PMI](https://www.pmi.org/pmbok-guide-standards/foundational/pmbok) | 项目管理知识体系 | 范围、进度、成本、风险、干系人等管理域。 |
| [Atlassian: Project Management](https://www.atlassian.com/work-management/project-management) | 项目管理实践 | 计划、协作、任务和项目可视化。 |
| [Asana: Work Breakdown Structure](https://asana.com/resources/work-breakdown-structure) | WBS | 把项目拆成可管理工作包。 |
| [Lucidchart: Critical Path Method](https://www.lucidchart.com/blog/how-to-use-critical-path-method) | 关键路径 | 找出决定项目总工期的最长依赖链。 |

## 敏捷和交付节奏

| 来源 | 主题 | 可复用要点 |
|---|---|---|
| [Scrum Guide](https://scrumguides.org/scrum-guide.html) | Scrum | Backlog、Sprint、透明度、检视和适应。 |
