## 6. 选型快速决策

```text
需要 Toast？ → Sonner
需要拖拽？ → dnd-kit（React）/ Vue Draggable Plus（Vue）
需要富文本？ → Tiptap（通用首选）/ Novel（AI风格）
需要动画？ → Framer Motion（React）/ Auto-Animate（跨框架）
需要图表？ → ECharts（功能全）/ Recharts（React简单）/ Tremor（仪表盘）
需要表格？ → TanStack Table（Headless）/ AG Grid（企业级）
需要虚拟列表？ → TanStack Virtual
需要文件上传？ → Uppy（功能全）/ Filepond（好看）
需要 PDF？ → react-pdf（渲染）/ @react-pdf/renderer（生成）
需要地图？ → Leaflet（免费）/ Mapbox GL（精美付费）
需要后台模板？ → vue-vben-admin（Vue）/ Ant Design Pro（React）
需要管理框架？ → Refine（React CRUD）/ React Admin
需要校验？ → Zod（首选）/ Valibot（极轻量）
需要工具函数？ → es-toolkit（现代）/ Radash（TS 友好）
需要 i18n？ → i18next（跨框架）/ vue-i18n（Vue）
需要日期？ → Day.js（轻量）/ date-fns（函数式）
```
