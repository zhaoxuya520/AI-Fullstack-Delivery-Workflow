# API 版本和兼容性指南

## 1. 兼容变更

通常兼容：

```text
新增可选响应字段
新增端点
新增可选请求参数
新增错误码但不改变原有错误语义
放宽校验规则
```

## 2. 不兼容变更

通常不兼容：

```text
删除字段
字段改名
字段类型变化
字段含义变化
必填规则变更
枚举值语义变化
错误结构变化
认证方式变化
```

## 3. 版本策略

```text
路径版本：/v1/resources
Header 版本：API-Version: 2026-05-18
媒体类型版本：application/vnd.example.v1+json
```

选择一种后保持一致。

## 4. 弃用策略

```text
公告时间
迁移指南
替代端点
迁移窗口
下线时间
监控受影响调用方
回滚策略
```

## 5. 变更检查

```text
是否影响旧客户端？
是否需要双写或双读？
是否需要兼容字段？
是否需要迁移文档？
是否需要通知第三方？
```
