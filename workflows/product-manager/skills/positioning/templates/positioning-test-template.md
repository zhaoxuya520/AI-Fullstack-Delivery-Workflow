# 定位声明测试模板

## 替换测试

把产品名替换成主要竞品，看声明是否还成立。

| 测试 | 替换为 | 是否还成立 | 说明 |
|------|--------|----------|------|
| 测试 1 | [竞品 A] | ☐ 是 ☐ 否 | |
| 测试 2 | [竞品 B] | ☐ 是 ☐ 否 | |
| 测试 3 | [手动方案] | ☐ 是 ☐ 否 | |

如果还成立 → 定位不够独特，需要细化

## 5 秒测试

把声明给一个不了解产品的人看 5 秒，问以下问题：

| 问题 | 答案 | 答对 |
|------|------|------|
| 这是给谁用的？ | | ☐ 是 ☐ 否 |
| 解决什么问题？ | | ☐ 是 ☐ 否 |
| 和别的工具有什么区别？ | | ☐ 是 ☐ 否 |

如果有任何"否" → 定位不够清晰
