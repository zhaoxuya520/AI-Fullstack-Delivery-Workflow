# 用户故事模板

## 用户故事格式

```text
作为 <用户角色>
我希望 <完成某个动作>
以便 <获得某个价值>
```

## 用户故事清单

| 编号 | 用户角色 | 用户故事 | 业务价值 | 优先级 | 备注 |
|---|---|---|---|---|---|
| US001 | | 作为...我希望...以便... | | Must | |

## 拆解维度

```text
角色：谁使用？
动作：做什么？
价值：为什么做？
触发：什么时候发生？
结果：完成后得到什么？
```

## 示例

```text
作为普通用户
我希望可以通过手机号登录
以便我不需要记住额外账号密码也能快速进入系统
```
