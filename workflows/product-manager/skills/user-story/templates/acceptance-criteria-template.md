# 验收标准模板

## Given / When / Then 格式

```gherkin
Given <前置条件>
When <用户行为或系统事件>
Then <可观察、可测试的结果>
```

## 验收标准清单

| 编号 | 关联功能 | Given | When | Then | 优先级 |
|---|---|---|---|---|---|
| AC001 | | | | | Must |

## 必须覆盖的场景

```text
成功路径
失败路径
空数据
权限不足
重复提交
边界输入
网络异常
数据冲突
```

## 不合格写法

```text
页面体验好
接口速度快
功能正常
系统稳定
```

## 合格写法

```text
Given 用户未登录
When 用户访问订单详情页
Then 系统跳转到登录页，并在登录后返回原订单详情页
```
