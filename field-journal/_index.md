## 高频踩坑

1. 需求没验收标准就进入开发，后续返工概率高。
2. API 字段没有契约，前后端联调会反复对齐。
3. 数据库迁移没有回滚方案，上线风险不可控。
4. 只跑单元测试不做核心路径验证，UI 和联调问题容易漏。
5. 部署没有健康检查和回滚方案，故障恢复时间不可控。
